  1. px :- Pixel (individual)
  2. viewport :- relative to screen display
    1. vw :- viewport width
    2. vh :- viewport height
  3. em :- relative to font size of parent container (if 2em = 2 *size of root element). if parent height = 10px then child height = 2 * 10 = 20
  4. rem :- relative to font size of root element (2rem = 2 * size of root element)
  5. vmin :-
    1. In computer vmin = height
    2. In phone vmin = width
  6. vmax:-
    1. In computer vmax = width
    2. In phone vmax = height
  7. min height :- min height of the container. (if for any containers min-height = 50px and height = 10px... for this case height of container will 50px)
  8. max height :- max height of the container. (if for any container max-height = 100px and height = 40px...for this case height of container will 100px)
  9. % :- n% of parent element. (if height of a container is 100px, height of child 80%, height of child will be - 80% of 100px = 80px)
